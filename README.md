# Event countdown

This project provides live timers for events entered in form. 

## How to run locally

`npm install` - install all dependencies  
`npm run dev` - run dev mode on port 5173

## Technologies

- React
- JavaScript
- HTML
- CSS
- Vite

## Functionalities

- Form
- Timer
- Validation form
- RWD
